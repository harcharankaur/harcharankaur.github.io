Designing a Wireframe using Photoshop

A website wireframe is a basic visual guide used in a site's interface design that sets its structure and helps maintain design consistency throughout your project. They are usually completed before any artwork is developed.

Follow the following steps:

  • • Create a background : With Photoshop open, create a new image by selecting File, New from the toolbar. Then create a custom dimension of 800 * 600 pixels.
  • • Design elements : Now we can start adding text boxes, image boxes, headers, footers, banners, drop-down menus, and linked buttons.
  • • Now it's time to add text descriptions to identify the purpose of each element we've added. Make sure your color palate selection will contrast with the background so the text will be visible.
  • • Next we will add sub-elements to further define the layout of the page
  • • As a web developer you will find that every project, client, and situation is different, but when you make the initial effort to lay the foundation of the website it will reap rewards in the end.

Designing a Responsive Design using HTML and CSS

Programming with HTML5 and CSS3 technologies have dramatically improved the landscape in web development. These technologies have certainly presented better ways to develop intriguing websites with unique interfaces.

Follow the following steps:

  • • Create a basic HTML file : The HTML5 embraces simple and easily understandable doctype. Moreover, it also eradicates the usage of self-closing tags, and introduced several new elements, including menu, video and audio, to name a few. It eventually saves a lot of web development time and efforts.
  • • Working on various sections of Page : Broadly, there are three different sections on a page, including header, content div and the footer. They have been defined in the code by their respective tags. The ‘nav’ tag has been also used to define the navigation elements. Through the section tag, the content section and sidebar are defined. In order to make them float, CSS style needs to be considered
  • • Styling via CSS3 : To enhance the basic layout created via the HTML in step 1 and create a refined look and feel, CSS3 can be used for styling in a required way. Here I have worked on CSS to create pretty effects.
  • • Creating a Responsive Blog using HTML5 and CSS3: For creating a responsive blog, one can use the attributes of the targeted device. However, there is a better solution that can be used by implementing a CSS framework like the Bootstrap.
